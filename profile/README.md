## The problem: Pixel Flow repeating levels (and why your “Level X” video doesn’t match)

Pixel Flow does this annoying thing: **the same level number can show different board layouts** for different players. AB tests, difficulty tuning, anti-walkthrough trolling. Whatever the reason, it makes “search level number → watch video” unreliable.

So we built the fix.

## Pixel Flow rules (the stuff that actually matters)

* **Color matching is strict.** A pig only hits cubes of its own color.
* **Ammo matters.** Big ammo pigs are traps if the board doesn’t have enough of that color exposed.
* You have **5 waiting slots**. Fill them and you’re basically dead.
* The board is layered. You often have to **peel outer colors** to expose the real targets. 

### Pro-tip (the “stop bleeding coins” trick)

When a run starts going bad, don’t “hope tap” your way out. **Restart early** the moment you can see you’ll need to park 3–4 pigs with leftover ammo. Saving a single slot is usually worth more than any random “continue” click.
